Horizontally and Vertically Center an Absolute Variable Block Element

Set top and left to 50% to find the middle of the space.

Since elements are positioned from the top, left corner we need to offset our reference. You can do this by using negative margins equal to half of the height and width.

However, in a variable height and width block, we do not know the height and width. We must rely on JavaScript to tell us the height and width of the block.
