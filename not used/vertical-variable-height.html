Vertically Center a Variable Height Block Element

A variable height block element can be vertically centered by using absolute positioning. Set top to 50% to find the middle of the space.

Since elements are positioned from the top, left corner we need to offset our reference. You can do this by using negative margins equal to half of the height and width.

However, in a variable height block, we do not know the height. We must rely on JavaScript to tell us the height of the block.
