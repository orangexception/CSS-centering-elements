Horizontally Center an Absolute Variable Width Block Element

That's a weird title. I mean a variable width block element that is absolutely positioned. Set left to 50% to find the middle of the space.

Since elements are positioned from the top, left corner we need to offset our reference. You can do this by using negative margins equal to half of the width.

However, in a variable width block, we do not know the width. We must rely on JavaScript to tell us the width of the block.
